# MVC - 2020

## Instalação (localhost)

Clonar o projeto
```bash
$ git clone git@github.com:diegovernan/mvc_2020.git
```

Instalar composer
```bash
$ composer install
```

## Configuração

Importar arquivo `mvc-2020.sql`

Configurar arquivo `src/Config.php`

## Utilização

Acessar `localhost/mvc-2020/public`
